CLOI Commands (암호화 페이지)

토큰 목록 — Copy 버튼을 눌러 토큰을 클립보드에 복사합니다. 토큰 문자열은 페이지에서 숨겨져 있습니다.

  • Debug Mode Change
    디버그 모드를 변경합니다. Level(minimal / verbose / full), Sample Rate(10~100%), auto_disable_hours(6~24)를 선택하여 적용할 수 있습니다.
  • Warmup test
    로컬 게이트웨이/런타임 헬스 체크(웜업)를 실행하여 서비스 준비 상태를 확인합니다.
  • Publish Webtoons (from Google Sheets)
    구글 시트에서 웹툰 목록을 읽어 정리된 HTML을 생성하고 GitHub Pages에 게시합니다.

암호를 암호화하려면 암호화 페이지로 이동하세요.
